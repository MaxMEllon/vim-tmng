## Installation

- Vundle

```vim
Bundle 'MaxMEllon/tmng-vim'
```

execute `:BendleInstall`

- NeoBundle

```vim
NeoBundle 'MaxMEllon/tmng-vim'
```

execute `:NeoBendleInstall`

- vim-plug

```vim
Plug 'MaxMEllon/tmng-vim'
```

execute `:PlagInstall`
